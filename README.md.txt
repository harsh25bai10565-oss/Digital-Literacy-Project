# Digital Literacy Project (CSE0001)
### Student Digital Ambassador: Harsh Gabrani
### VIT Bhopal University

## Project Overview
This repository contains the complete set of tasks for the Digital Literacy course. The project focuses on building essential digital skills for modern academic and professional environments.

## Repository Contents
* **Task 1: Infographic** - Visual guide on Digital Literacy and safe internet practices.
* **Task 2: Portfolio** - Professional profiles on GitHub, LinkedIn, and Kaggle.
* **Task 3: Platforms** - Coding proficiency on HackerRank and collaborative tools via Google Forms.
* **Task 4: Etiquette** - Professional email drafts and social media Do's & Don'ts.
* **Task 5: Cybercrime** - Case study on UPI/Payment fraud and a prevention checklist.
